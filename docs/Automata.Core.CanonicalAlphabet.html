    Class CanonicalAlphabet

    Immutable and optimized representation of an alphabet with contiguous, ordered symbols.

    Remarks

    A CanonicalAlphabet is defined by the following properties:

    1. Immutable: Structural and behavioral invariance, ensuring thread safety and predictable behavior.
    2. Performance: Optimized for fast, read-only operations with minimal memory overhead.
    3. Ordering: Symbols are sorted according to CanonicalStringComparer.
    4. Indexing: Symbols are associated with contiguous, ordered integer indices [0..Count).
