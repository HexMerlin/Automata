    int
    Remarks

    This value denotes an upper bound for the state numbers in the DFA. The actual maximum state number may be lower (but not higher), since we do not keep track of removed states for performance reasons.

    Accepts(IEnumerable<string>)

    Indicates whether the DFA accepts the given sequence of symbols.

    Declaration
    public bool Accepts(IEnumerable<string> sequence)
    Parameters
    Type Name Description
    IEnumerable<string> sequence

    The sequence of symbols to check.
    Returns
    Type Description
    bool

    true if the DFA accepts the sequence; otherwise, false.
    Remarks

    The DFA processes each symbol in the sequence, transitioning between states according to its transition function. If the DFA reaches a final state after processing all symbols, the sequence is accepted.
