Remarks

ImmutableTransitions is characterized by:

  • Immutable: Structural and behavioral invariance, ensuring thread safety and predictable behavior.
  • Performance: Optimized for fast, read-only operations with minimal memory overhead.
  • Ordering: Transitions are sorted according to the default order of Transition.
  • Deterministic: The set cannot contain more than one transition with a specific (FromState, Symbol) pair.
